# Инструкция по установке Dst-Recorder на macOS

## Проблема: "Приложение повреждено"

После скачивания DMG файла macOS может показать сообщение:
> Приложение «Dst-Recorder.app» повреждено, и его не удается открыть. Переместите приложение в Корзину.

Это происходит потому, что приложение не подписано сертификатом Apple Developer.

## Решение

### Способ 1: Через терминал (рекомендуется)

1. Откройте DMG файл и перетащите `Dst-Recorder.app` в папку `Applications`
2. Откройте **Терминал** (Finder → Программы → Утилиты → Терминал)
3. Выполните команду:
   ```bash
   xattr -cr /Applications/Dst-Recorder.app
   ```
4. Теперь можете запустить приложение из папки `Applications`

### Способ 2: Через System Settings

1. Попробуйте открыть приложение (появится сообщение об ошибке)
2. Откройте **System Settings** (Системные настройки)
3. Перейдите в **Privacy & Security** (Конфиденциальность и безопасность)
4. Прокрутите вниз до раздела **Security**
5. Нажмите кнопку **Open Anyway** (Открыть в любом случае) рядом с сообщением о Dst-Recorder
6. Подтвердите открытие приложения

### Способ 3: Через контекстное меню

1. В Finder найдите приложение `Dst-Recorder.app`
2. Удерживая **Control**, кликните на приложение (или кликните правой кнопкой мыши)
3. Выберите **Открыть** в контекстном меню
4. В диалоге подтверждения нажмите **Открыть**

## После первого запуска

При первом запуске macOS попросит разрешения:
- **Запись экрана** - для захвата видео
- **Микрофон** - для записи звука

Предоставьте эти разрешения в System Settings → Privacy & Security.
